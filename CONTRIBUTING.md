## Access Restrictions

This project is **NOT** open source. Contributing to this project requires:

1. **Explicit written authorization** from the project owner
2. **Signed confidentiality agreement** (NDA)
3. **Acceptance of proprietary license terms**

## For Authorized Contributors

If you have been granted authorization to contribute:

### 1. Confidentiality

- All code, documentation, and discussions are confidential
- Do NOT share any project information publicly
- Do NOT discuss project details outside authorized channels
- Keep all credentials and access tokens secure

### 2. Development Process

1. Create a feature branch from `main`
2. Make your changes following project coding standards
3. Test thoroughly (backend: `pytest`, frontend: `npx vitest run`)
4. Submit a pull request with clear description
5. Address code review feedback
6. Wait for approval before merging

### 3. Code Standards

- **Python**: Follow PEP 8, use type hints
- **TypeScript**: Use strict mode, follow ESLint rules
- **Testing**: Maintain or improve test coverage
- **Documentation**: Update docs for any changes

### 4. Commit Guidelines

- Use clear, descriptive commit messages
- Reference issue numbers where applicable
- Keep commits focused and atomic

### 5. Intellectual Property

**By contributing, you acknowledge that:**

- All contributions become the exclusive property of PilotForge
- You assign all rights, title, and interest in your contributions
- You have the right to make the contribution
- Your contribution does not violate any third-party rights

### 6. Security

- Report security vulnerabilities privately (see SECURITY.md)
- Never commit secrets, API keys, or credentials
- Use environment variables for sensitive data
- Follow secure coding practices

## Unauthorized Access

If you have accessed this repository without authorization:

1. **STOP** - Close this repository immediately
2. **DELETE** any local copies you may have created
3. **DO NOT** share or discuss any information you may have seen
4. **CONTACT** the repository owner if you believe you received access in error

Unauthorized access, use, or disclosure of this proprietary software may result in:
